A complete backend system built with Spring Boot + MySQL, designed for an e-learning platform that includes:

✔ Course content management (courses, chapters, videos)
✔ User authentication (JWT-based login & signup)
✔ Quiz system with random questions
✔ Admin panel features
✔ User progress tracking
✔ Logging, validation, exception handling
✔ Postman collection for full API testing

This project demonstrates enterprise-level backend architecture suitable for production use.
